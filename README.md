# Portal

My personal webpage built with [Gatsby](https://www.gatsbyjs.com/), [Tailwind CSS](https://tailwindcss.com/), and TypeScript.

This was created because I wanted to dabble in front-end technologies, as well as create my own webpage and blog.

## Why did you choose...

- [Gatsby](https://www.gatsbyjs.com/): I wanted to examine how the developer experience differs from [Next.js](https://nextjs.org/).
- [Tailwind CSS](https://tailwindcss.com/): I wanted to examine the pros and cons of their utility-first approach to CSS.

## Getting started

Run this locally on `http://localhost:8000`:

```
npm run dev
```
